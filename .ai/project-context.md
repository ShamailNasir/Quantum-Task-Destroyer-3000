Project Name: AI Productivity App

VISION:
A modern, intelligent productivity system that helps users manage tasks, track progress, and improve efficiency using AI.

CORE IDEAS:

- Task management system
- Clean dashboard UI
- Gamification (XP, levels)
- AI-based suggestions
- Offline-first capability

TECH STACK:

Frontend:
- Next.js (React framework)

Backend (later):
- FastAPI (Python)

Storage Plan:
- Phase 1: LocalStorage
- Phase 2: Database (PostgreSQL or similar)

IMPORTANT NOTES:

- Features are flexible and can change anytime
- UI is not final and can evolve
- System must remain scalable
- Code should stay clean as features grow

GOAL:

This is not a demo project.
It should feel like a real product.
