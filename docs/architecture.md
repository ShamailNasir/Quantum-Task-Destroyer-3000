# Architecture Overview

This project is divided into two main parts:

1. Frontend (Next.js)
   - Handles UI
   - User interaction
   - Client-side logic

2. Backend (FastAPI) [Later Phase]
   - API handling
   - Database operations
   - AI processing

Development Strategy:

- Start with frontend only
- Use localStorage for data
- Introduce backend later

Goal:

Build a scalable system step by step without overcomplicating early stages.
